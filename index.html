About this project

Source Code
Language: JavaScript
Framework: HTML Canvas

A sorting program. Includes a graph of adjustable size and a GUI made with a custom button class. Ten different sorting algorithms are available to choose from. The sorting process is done with individual frames for each array operation. After each sort is complete, runtime statistics are displayed.
